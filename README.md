# Pomodoro-番茄鐘
- 一個小作品demo
- 使用html/javascript/jQuary/css/chart.js


## 主要功能
- 番茄鐘加上待辦清單與統計

- 番茄鐘，demo部分時間為五秒工作時間與3秒休息時間輪流
![](https://github.com/jc33654/Pomodoro/raw/master/demo_img/1.png)

- 待辦清單，可新增待辦事項
![](https://github.com/jc33654/Pomodoro/raw/master/demo_img/2.png)

- 番茄鐘數量統計，統計每週番茄數量與每日數量並加上圖表
![](https://github.com/jc33654/Pomodoro/raw/master/demo_img/3.png)
